# Tableau Export Guide

Use this structure for GitHub-ready output images.

## Folders

- output/tableau/worksheets
- output/tableau/dashboards

## Worksheet Export Naming

1. 01_kpi_cards.png

## Tableau Steps

For each worksheet/dashboard:
1. Open the sheet/dashboard.
2. Go to Worksheet or Dashboard menu.
3. Click Export -> Image.
4. Save into the correct folder above.

## GitHub Tip

Keep image width consistent for better README display.
Recommended width: 1366 px.
